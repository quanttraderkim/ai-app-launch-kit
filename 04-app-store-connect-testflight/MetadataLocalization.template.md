# Metadata Localization Template

App Store Connect 메타데이터는 처음부터 완벽할 필요는 없지만, 앱 정체성과 테스트 목적이 분명해야 합니다. 이 파일은 한국어와 영어를 같이 준비할 때 사용할 수 있는 템플릿입니다.

## Korean

```text
Name: <KO_APP_NAME>
Subtitle: <KO_SUBTITLE>
Promotional Text: <KO_PROMOTIONAL_TEXT>
Description:
<KO_DESCRIPTION>

Keywords: <KO_KEYWORDS_COMMA_SEPARATED>
Support URL: <SUPPORT_URL>
Marketing URL: <MARKETING_URL_OPTIONAL>
```

## Review Checklist

앱 이름은 임시 프로젝트명이 아니라 사용자가 이해할 수 있는 이름이어야 합니다. 키워드는 과장하거나 경쟁사 상표를 남용하지 않습니다. 설명은 기능 목록보다 사용자가 언제 왜 쓰는지 먼저 말합니다.
