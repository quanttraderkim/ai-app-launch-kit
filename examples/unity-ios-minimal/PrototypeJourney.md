# Prototype Journey: 탭 전투에서 빌드 선택까지 (2주, 19빌드)

스터디 킥오프용 요약. 본게임 전에 돌린 준비 프로토타입(픽셀 자동 크롤러)이
**무엇을 검증했고, 그 노하우가 kit 어디에 있는지** 30분 안에 훑을 수 있게 정리했다.

## 본게임에 그대로 이식할 검증된 시스템

1. **전투/성장 코어**: 순수 C# 모델(HP·반격·킬힐·드랍) + 스테이지 리듬(킬 목표→정비→진행).
2. **경제**: 킬 보상 명시(HUD+배너) + 소비 3선택(강화 비용제/회복/진행) + 패배 후에도 소비처(Regroup).
3. **빌드 선택**: 장비 kind=스타일(공격 4×방어 3), 탭 장착, opt-in 원칙.
4. **컷신**: 대본=struct 데이터, 재생 기계 1개(타이핑/선택지/1회성), 지역·상황별 대본 선택 함수.
5. **검증 인프라**: 배치 컴파일 → sim 스크린샷 → autoplay 딥런(정비·컷신 자동 탭) → TestFlight 실기.
