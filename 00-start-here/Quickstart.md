# Quickstart

이 문서는 새 멤버가 AI 에이전트와 앱 프로젝트를 시작할 때 처음 45분 동안 할 일을 정리한 것입니다. 목표는 완벽한 기획이 아니라, 첫 주 안에 “무엇을 만들고, 어떻게 검증하고, 어떤 환경이 필요한지”를 분명히 하는 것입니다.

## First 45 Minutes

1. 이 저장소를 클론합니다.
2. 에이전트에게 `README.md`, `AGENTS.md`, `01-agent-operating-system/AgentPrompt.md`를 읽게 합니다.
3. 앱 아이디어를 한 문장으로 씁니다. 예: “퇴근길 3분 동안 오늘의 감정을 기록하고 다음 행동 하나를 추천받는 앱.”
4. `02-product-prd/PRD.template.md`를 복사해서 내 프로젝트의 `Docs/PRD.md`로 만듭니다.
5. `02-product-prd/Iteration.template.md`를 복사해서 첫 iteration 후보를 만듭니다.
6. `03-ios-unity-setup/EnvironmentChecklist.md` 또는 내 플랫폼에 맞는 체크리스트를 채웁니다.
7. Apple 배포가 목표라면 `04-app-store-connect-testflight/AppStoreConnectSetup.md`와 `04-app-store-connect-testflight/TestFlightRunbook.md`를 읽고, 계정 준비 상태만 먼저 표시합니다.

## First Agent Prompt

```text
이 repo를 기준으로 내 앱 프로젝트를 시작해줘. 먼저 PRD와 첫 iteration 문서를 만들고, 내 개발환경과 TestFlight 준비 상태를 점검해줘. 실제 계정값이나 비밀키는 문서에 쓰지 말고 placeholder로 남겨. 첫 proof가 너무 커지면 범위를 줄여줘.
```

## Done Criteria For Day 1

Day 1이 끝날 때 필요한 것은 거대한 앱이 아니라 명확한 방향입니다. `Docs/PRD.md`가 있고, 첫 proof 범위가 있고, 개발환경 blocker가 정리되어 있고, Git repo에 첫 커밋이 있으면 충분합니다.
